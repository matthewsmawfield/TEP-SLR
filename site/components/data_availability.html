Data Availability

Primary Data Sources

SLR Observations: All Satellite Laser Ranging data used in this analysis are publicly available from the International Laser Ranging Service (ILRS) via NASA's Crustal Dynamics Data Information System (CDDIS):

  • Normal Point Data: https://cddis.nasa.gov/archive/slr/data/npt_crd/ (DOI: 10.5067/SLR/slr_data_monthly_npt_001)
  • Satellite Orbits (SP3): ILRS Combination Center products available at https://cddis.nasa.gov/archive/slr/products/orbits/ (DOI: 10.5067/SLR/SLR_ILRSORB_001)
  • Station Coordinates: SLRF2020 reference frame (DOI: 10.5067/SLR/slrf2020_001)

Analysis Code & Processed Data

GitHub Repository: All analysis scripts, processing pipelines, and intermediate data products are available at:

https://github.com/matthewsmawfield/TEP-SLR

The repository includes:

  • scripts/steps/step_2_1_slr_residuals.py — Primary residual computation pipeline
  • scripts/steps/step_2_3_mwpc_analysis.py — Inter-station correlation analysis
  • scripts/helpers/process_residuals_yearly.py — Spectral and temporal diagnostics
  • results/outputs/ — JSON summary files with all reported statistics
  • reproduce_analysis.sh — One-command reproduction script

Archived Dataset

Zenodo Archive: A complete snapshot of the processed dataset, analysis outputs, and figures is permanently archived at Zenodo:

DOI: 10.5281/zenodo.18064582

This archive contains:

  • Filtered residual time series (192,561 high-precision observations)
  • Station-pair correlation matrices and distance-binned statistics
  • Power spectral density estimates for all 46 stations
  • Bootstrap confidence interval distributions
  • All figures in publication-quality format (PNG, PDF)

Reproducibility

To reproduce the analysis from raw ILRS data:

  1. Clone the repository: git clone https://github.com/matthewsmawfield/TEP-SLR.git
  2. Install dependencies: pip install -r requirements.txt
  3. Run the pipeline: bash reproduce_analysis.sh

The pipeline will download ILRS data, compute residuals, perform all statistical tests, and regenerate figures. Total runtime: ~2-4 hours on a standard workstation.

Software Dependencies

Analysis performed using Python 3.10+ with the following key packages:

  • NumPy 1.24+ (numerical computation)
  • SciPy 1.10+ (statistical tests, spectral analysis)
  • Pandas 2.0+ (data manipulation)
  • Matplotlib 3.7+ (visualization)

Full version specifications in requirements.txt.
